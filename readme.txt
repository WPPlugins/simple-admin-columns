Add modified date sortable column, featured image custom column to the admin interface for post and pages.

== Description ==

Simple Admin Columns is very easy to use WordPress plugin. It helps you to create `custom columns` on the admin screens for posts and pages.

You can create a `Modified Date` sortable column and `Post Thumbnail` column for posts and pages in WordPress admin area by using a friendly interface.

= Features =
* Simple Admin Columns is very easy to use.
* You can create a sortable `Last Modified` or `Modified Date` column on posts and pages screen.
* You can add `Featured Post Thumbnail` column on posts and pages screen.

= About =
Simple Admin Columns plugin is developed by [WPFlask](https://wpflask.com/). You can use our free WordPress themes for your blog.

== Installation ==

1. Upload the `wpflask-admin-columns` folder to the `/wp-content/plugins/` directory.
1. Activate the `Simple Admin Columns` plugin through the 'Plugins' menu in WordPress.
1. Configure the plugin by going to the `Settings > Simple Admin Columns` options.

== Screenshots ==

1. Simple Admin Columns Options: You can control the display of custom columns by using the simple options.
2. `Last Modified` Sortable Column: You can create `Last Modified` or `Modified Date` sortable column in posts and pages admin interface.
3. `Post Thumbnail` Column: You can create `Featured Image` or `Post Thumbnail` column in posts and pages admin screen.
